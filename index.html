This page is for Java Script language practice

To see the result inspect the web page and go to console or you can see it at editor terminal section

to check the instellation of node js type 'node -v' at editor terminal

to see the js result at terminal type 'node (js file name)' exm: "node js1_basic_js_and_arithmetic_operation.js"

for auto watch need to install "nodemon" at terminal. To install type "npm i nodemon". Then go to package.json and add a script

"script": {
"start": "nodemon (js file name)"
}

then type to the terminal "npm run start"
